🏋️ Sistema de Academia - Java POO
📌 Descrição

Projeto desenvolvido com o objetivo de praticar os conceitos de Programação Orientada a Objetos (POO) em Java, simulando um sistema simples de gerenciamento de academia.

🚀 Funcionalidades

Cadastro de alunos

Alteração de plano (Mensal, Trimestral, Anual)

Adição de treinos aos alunos

Criação de treinos por instrutores

Atribuição de treinos aos alunos

Remoção de alunos

Listagem de alunos cadastrados

🧠 Conceitos aplicados

Encapsulamento

Herança (Pessoa → Aluno, Instrutor)

Polimorfismo (toString)

Uso de Enum

Separação de responsabilidades (Service)

Manipulação de listas (simulando banco de dados)

🛠️ Tecnologias utilizadas

Java 8+

Programação Orientada a Objetos
